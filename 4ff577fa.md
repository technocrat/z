# decimal format

```r
A more general function is as follows where x is the number and k is the number of decimals to show.

    specify_decimal <- function(x, k) format(round(x, k), nsmall=k)

    specify_decimal(1234, 5)
    [1] "1234.00000"
    specify_decimal(0.1234, 5)
    [1] "0.12340"

```
